# JogaTeraTech Landing Page - NextJs

This is a web front-end and back-end source code for JogaTeraTech Landing Page.

## Installation and Run Locally

Clone the project with SSH

```bash
  git clone git@gitlab.com:Jogateratech-organization/jogateratech.git
```

Go to the project directory

```bash
  cd jogateratech
```

Install dependencies with npm

```bash
  bun install --save
```

Start the server

```bash
  bun run dev
```
